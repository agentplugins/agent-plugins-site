# Open Plugins Docs

Documentation site for [Open Plugins](https://open-plugins.com) — a standard for packaging AI coding agent extensions into distributable plugins. Built with Next.js and [Fumadocs](https://fumadocs.dev).

## Features

- MDX-powered documentation with full component support
- AI-powered chat that understands the Open Plugins specification
- GitHub Discussions integration for user feedback
- Modern, accessible UI built with Radix UI
- Fast, fuzzy search through all documentation
- Dark mode with built-in theme switching
- Responsive, mobile-first design
- Built on Next.js 16 with App Router
- RSS feed for documentation updates

[Read the docs](https://open-plugins.com) to get started.
